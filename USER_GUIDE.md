# AquaShield 智慧水產養殖數據中心 - 使用者操作手冊

本手冊介紹 **AquaShield 智慧水產養殖數據中心** 的各項核心功能與操作指引。本平台整合了 **IoT 實體串接**、**3D 視覺化**、**WebAR** 與 **OpenAI 智慧助理** 技術，旨在提供一站式的智慧養殖監控體驗。

---

## 🗺️ 系統版面與核心功能說明

### 1. 左側邊欄：魚池管理區
- **魚池列表卡片**：
  顯示當前所有的魚池（例如：`Pond Alpha (虱目魚)`）。卡片內有動態運轉的水車圖標與指示燈，方便您即時掌握水車是否在運作。
- **新增魚池**：
  點選魚池列表底部的 **`+ 新增魚池` 虛擬卡片**，即可開啟對話框，輸入名稱與地點，快速新增一個魚池。新增後系統會自動為該魚池配置四個預設的感測器（溫度、pH、溶氧量、水位）。
- **選取切換**：
  點選任何一個魚池卡片，整個數據中心（包括 3D 模型、感測讀數、歷史圖表、水車控制）將會同步切換為該選定魚池的即時資料。

---

### 2. 中央區域（上）：3D 即時模擬與感測監控

#### 🎬 3D 魚池模擬器
- 利用 **Three.js** 渲染出的虛擬圓形魚池。
- **動態魚群與多生物模擬**：
  - 點選 3D 畫面右上角的選單，可以自由選取當前模擬的生物種類：
    - 🐟 **魚類**：流線型金橘色魚身，搭配精細的胸鰭、背鰭，尾鰭會逼真地左右擺動悠游。
    - 🦐 **蝦類**：半透明粉紅色身體與長鬚，游動足（腹足）會划水前進，並伴隨蝦子特有的「倒退彈跳」避敵動作。
    - 🦀 **螃蟹**：扁平的硬殼、大螯與六隻步足，會在魚池底部（池底層）以寫實的橫向步伐側行爬動。
- **水車與氣泡**：若該池有任何水車處於「運轉中」狀態，3D 畫面的水車便會快速旋轉，並產生源源不絕的氰藍色曝氣氣泡，視覺化呈現溶氧效果。
- **運轉率顯示**：在控制中心標題旁會即時計算並顯示當前水車的 `運轉率: XX%`。3D 模擬中的水車旋轉速度與氣泡產生率，將會與此運轉率成正比（例如 1/2 台運轉時速度為 50%）。
- **實體感測器串接**：
  點選 3D 面板下方的 **`實體感測器串接`** 按鈕，會彈出 API 文件指引。您可以根據其中的 **Python 測試腳本** 或 **ESP32 Arduino 程式碼範例**，編寫實體硬體裝置，透過 HTTP POST 將現場感測數據上報至本平台。
- **AR 實境投影**：
  點選 **`AR 實境投影`** 按鈕，會啟動相機權限，投影一個太空科幻風格的 Heads-Up-Display (HUD) 抬頭顯示器。若在手機或支援的設備上，可模擬現場巡檢時將魚池狀態虛擬疊加於真實水池的 AR 效果。

#### 📊 感測即時數據格（Telemetry Grid）
- **即時數據卡**：顯示該魚池當前的「水池溫度」、「酸鹼 pH 值」、「水中溶氧量」與「目前水位」。
- **新增感測器**：
  在感測器卡片旁邊，設有 **`+ 新增感測器` 虛擬卡片**，點選即可快速為當前魚池新增額外的感測器項目。

#### 🌀 增氧水車控制中心與智慧自動化
- **個別水車控制**：
  列出該魚池內所有的水車，點選開關即可控制單台水車。您也可以點選 `新增水車` 或點選垃圾桶圖示 `刪除` 水車。
- **🤖 智慧溶氧自動控制（自動化迴路）**：
  - **自動化開關**：開啟後，系統會啟動反饋監控。
  - **臨界溶氧值**：當水中溶氧量低於此設定值（例如：`4.0 mg/L`）時，系統會**自動開啟**該池所有水車進行曝氣；當溶氧量回升並穩定後，會**自動關閉**水車以節省電費。
  - 修改數值後點選 **`套用`** 即可儲存設定。

### 3. 中央區域（下）：歷史監測數據與 AI 水質健檢
- **歷史趨勢圖表**：
  您可以自由篩選想要檢視的「感測器項目」與「時間範圍」（24 小時、7 天、30 天），系統會使用 Chart.js 畫出色彩亮麗的折線趨勢圖。
- **表格視窗切換**：
  可切換為「詳細數據列表」，以表格形式閱讀每筆時間戳記與數值。
- **AI 水質健康動態分析評級**：
  系統會根據歷史讀數進行運算，給出當前魚池的健康評等（例如：正常、警告、危險），並提供針對性的養殖建議（如：建議增加換水頻率、注意溶氧不足等）。

---

### 4. 右下角：智慧養殖 AI 助理（OpenAI 驅動）
- 點選右下角的 **🤖 機器人圖標** 即可開啟聊天對話框。
- **自然語言互動**：您可以直接與 AI 對話，例如：
  - *"目前有多少個魚池？"*
  - *"幫我新增一個名稱為 '南區生態池' 的魚池，位置在 '南區 02'"*
  - *"幫我關閉 Pond Alpha 的水車"*
  - *"如果 Pond Beta 的溶氧量太低，幫我開啟它的自動增氧並設定臨界值為 4.5"*
- **無縫 UI 連動**：當 AI 助理成功執行您的指令（例如幫您關閉水車或新建魚池）時，**網頁前端會完全自動更新**（3D 模型、水車開關按鈕、側邊欄列表），無需手動重新整理網頁！
